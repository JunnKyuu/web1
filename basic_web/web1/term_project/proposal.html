Term Project 3조 기획서

팀장: 이준규
조원: 고선제, 이기택, 이서현, 홍정우

  • 충족되어야 하는 목표, 기획 작성 책임자 식별

    • 목표
      • 각 선택을 버튼으로 구현하고 사용자의 선택을 입력받아 관련된 결과를 출력하는 웹페이지
      • 예를들어, 사용자의 선택을 입력받아 사용자의 성향과 맞는 개발 직무, 개발 언어 같은 것들을 출력
    • 책임자
      • 기획서 초안 작성: 조원 및 팀장 - 고선제, 이기택, 이서현, 이준규, 홍정우
      • 기획서 HTML 작성: 팀장 - 이준규

  • 달성 대상의 목표와 범위

    • HTML을 사용하여 사용자에게 보이는 기본적인 웹페이지 뼈대 만들기
    • CSS를 사용하여 HTML과 HTML 요소의 서식을 지정
      • 완성도 있는 UI 만들기
    • Javascript를 사용하여 동적으로 화면 구성
      • 사용자가 클릭시 다음 페이지 정상 로드
      • 사용자가 선택한 값에 알맞은 결과 출력

  • 충족여부 기준

















  • 제약사항 또는 위험

    • 동일 계산 값이 발생하지 않도록 설계
    • 모바일 친화적(View Port)
    • 선택 도중 웹페이지를 나가거나 새로고침할 경우, 선택한 정보 clear

  • 수행해야 하는 작업 및 일정

    • 프로젝트 기획
      • 주제 선정 - 9주차
      • 스토리보드 작성 - 10주차
    • 웹페이지 설계
      • HTML 작성 - 11주차, 12주차
      • CSS 작성 - 11주차, 12주차
      • 필요한 기능 JS로 구현 - 13주차, 14주차
    • 웹페이지 배포
      • Netlify를 통한 배포 - 15주차

  • 계획 달성에 필요한 방법/접근법

    • HTML과 CSS를 통한 프론트 구현
    • JS를 통한 서버 및 기능 구현

  • 계획 목표를 달성하기 위한 자원 (시간/인원/재료/장비/예산)

    • 시간: 개인 당 주 5시간, 회의 주 2시간
    • 인원: 5명
    • 재료: 여러 참고 자료들
    • 장비: 노트북, 마우스, 충전기 등등
    • 예산: 없음

  • 팀원 역할

    • 자료조사(어떤 내용 선택, 기능 구현을 위한 프로그래밍 지식)
    • 디자인 조사(웹페이지 디자인 예시 조사)
    • 고선제 - 출력 웹페이지(HTML, CSS), 자료조사
    • 이기택 - 도입 웹페이지(HTML, CSS), 자료조사
    • 이서현 - 출력 기능(JS), 자료조사, 디자인 조사
    • 이준규 - 팀장, 기획서 작성, 회의록 작성, 선택 웹페이지(HTML, CSS), 자료조사
    • 홍정우 - 선택 기능(JS) DOM, 자료조사
